# vue2.6.12学员管理

#### 介绍
项目基于 Vue2.6.12 开发的学员教师管理系统,采用RESTFUL架构风格,模块化开发模式,教师学员管理系统采用数据化管理学员信息和教师信息,该项目为针对学校学生、教师、专业进行统一管理的使用平台，因为学生较多而且分散的班级及授课教师比较杂，不方便信息查询以及管理，所以将所有学生及教师等集中到一个平台上方便快速查询学生信息。
